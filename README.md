# 📡 Featured Project

## UAV Antenna Tracker

A **Raspberry Pi powered antenna tracker** that automatically aligns an antenna toward a UAV using MAVLink telemetry.

### Core Features

- Real-time MAVLink telemetry tracking
- Compass based heading estimation
- Stepper motor precision control
- GUI based control interface
- Limit switch safety mechanism
- Smooth tracking algorithms for stable antenna movement

### Future Improvements

- Fully operational **dual axis tracking system**
- Predictive tracking algorithms
- Kalman filter based smoothing
- ROS based mocap tracking integration
